# Python-for-machine-learning
Les techniques les plus importantes vous sont présentés: Structures de données, Algorithmes, Programmation orientée objet, et pleins d'autres techniques python utile en Data Scienc
D'autres librairies utiles (et moins connues) vous seront enseignées : Seaborn pour la visualisation de données (tres utile en Data Science) et Scipy pour le calcul scientifique avancé.
